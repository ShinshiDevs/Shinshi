# Shinshi
Modern and user-friendly Discord bot designed to provide you and your servers with great features and stable performance. Its main purpose is to assist you in creating your own Discord server, diversify any server, and enhance your everyday experience.

Starting in 2021, the Shinshi project was launched, but it was halted due to lack of resources and motivation. However, in 2023, the project was revived and is being actively promoted. Over the years, the concept of Shinshi has remained unchanged - to create a stable and modern Discord bot that stands out from others with its unique approach.

# Stack
- The programming language used is [Python](https://python.org), CPython implementation.
- We employ the [hikari](https://hikari-py.dev) library in conjunction with our [Aurum](https://shinshidevs.github.io/aurum-hikari) framework to interface with the Discord API.
- We are considering the use of Pony ORM for database interactions with PostgreSQL.
- For code linting purposes, using tools as ruff and mypy.
