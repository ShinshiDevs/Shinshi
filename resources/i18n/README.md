# Shinshi Internationalization and localization

All localization files must have the same keys and that are not used in the bot itself.

They must have the format `.yaml` and have the same name as hikari.Locale of the language for which this file is being made.

This is due to simplification and in order to have a common "protocol" with hikari, how to get the desired localization and not make unnecessary construction to get it.
