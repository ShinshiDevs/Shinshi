### All icons must be
- must be with `.webp` extension.
- must be in `256x256` resolution.
### Logic
Icons can be placed in sub-folder, for example:
- if we've a command with many icons. `(folder name == command name)`
- if we've a module with many icons. `(folder name == module name)`

For now, the function for getting icons it not available.

> We use [Feather icons](https://github.com/feathericons/feather) and [Unicons](https://github.com/Iconscout/unicons).
